# Datatypes 

*Char*  Char is using 50 bytes of memory no matter what is the length of the sting is .So it is not efficient and wastage of memory

*VARCHAR*  VARCHAR is used for the variable length

*DATE*   Date in format of YYYY-MM-DD 
*BIT*   BIT(x) will store the BIT of x length 
*BOOLEAN*  BOOLEANS VALUES 0 or 1 
*SIGNED* means it takes positve and negative 
*UNSIGNED* means it takes only positives (like age)
